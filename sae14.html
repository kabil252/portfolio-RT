Introduction

Dans le cadre de cette SAÉ, j’ai développé un site web de vente de fleurs nommé Fleur Boutique.

Le projet contient plusieurs pages HTML reliées entre elles avec une feuille de style CSS afin de créer une interface moderne, élégante et responsive.

Le site permet d’afficher différents produits floraux avec des images, des descriptions et des prix.

Apprentissages Critiques (AC)

  • AC13.01 – Utiliser un système informatique et ses outils
  • AC13.02 – Modifier et adapter un programme ou une page web
  • AC13.03 – Structurer une page web en HTML
  • AC13.04 – Mettre en forme une interface avec le CSS
  • AC13.05 – Organiser les ressources et les fichiers d’un projet web

Ces apprentissages critiques ont été mobilisés durant cette SAÉ à travers la création du site web, l’organisation des ressources HTML/CSS et la résolution des problèmes liés au chargement des fichiers.

Réalisation

Pour réaliser ce projet, j’ai utilisé les langages HTML et CSS afin de créer l’interface du site web.

J’ai conçu plusieurs sections comme : l’accueil, les produits, la page de détails, les avis clients et la page contact.

J’ai également ajouté des images, des boutons interactifs et une navigation entre les pages.

Problème rencontré

Preuve 1 : HTML

Une image ne s’affichait pas sur le site web

Lors des tests du site web, une image produit ne s’affichait pas correctement dans la section des produits.

À la place de l’image, le navigateur affichait une image cassée ainsi qu’un espace vide dans la carte produit.

Analyse du problème

Après analyse du code HTML, j’ai constaté qu’une erreur était présente dans le chemin de l’image utilisé dans la balise HTML.

Le nom du fichier indiqué dans le code ne correspondait pas exactement au nom réel de l’image présente dans le dossier du projet.

Le problème provenait donc d’une erreur dans le chemin d’accès de l’image.

Résolution du problème

Pour corriger cette erreur, j’ai vérifié le contenu du dossier des images puis corrigé le nom du fichier dans le code HTML.

Après cette correction, l’image s’est affichée correctement dans la page des produits.

Preuve 2 : CSS

Utilisation du CSS dans le projet

Durant ce projet Fleur Boutique, j’ai utilisé un fichier CSS afin de personnaliser l’apparence du site web.

Le fichier CSS permettait d’ajouter les couleurs, les espacements, les animations et la mise en page du site.

Erreur de liaison du fichier CSS

Après une modification du fichier HTML, le style CSS ne se chargeait plus correctement.

Le site web apparaissait sans couleurs et sans mise en page.

Analyse du problème

Pour analyser le problème, j’ai vérifié le lien du fichier CSS dans la balise link du fichier HTML.

J’ai constaté qu’une erreur était présente dans le nom du fichier CSS. 

<link rel="stylesheet" href="styles.css">

Alors que le vrai fichier du projet s’appelait : 

style.css

Résolution du problème

Pour corriger cette erreur, j’ai modifié le nom du fichier CSS dans le code HTML. 


<link rel="stylesheet" href="style.css">

Après cette correction, le style CSS s’est chargé correctement et le site web a retrouvé son apparence normale.
